> 技术选型时整个业务开发所做的第一个项目,在正常时开发开发前会经过需求会议来决定项目的
>- 成品具体功能
>- 成品具体类别
>- 成品具体类别
>- 后续的制作程度

通过以上的信息来决定了`技术选型`,使用与哪些技术来完成所做出来的效果
